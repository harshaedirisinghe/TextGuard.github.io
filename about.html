About TextGuard

Vision

Our vision is to provide users with real-time monitoring, enhancing their defense against smishing attacks and promoting a proactive approach to cybersecurity.

Mission

Our mission is to provide a secure digital environment by leveraging machine learning techniques and integrating them into a user-friendly mobile application that empowers users to recognize and report suspicious text messages.


End Goals

To provide a machine learning-based system integrated into a mobile application for effective detection and prevention of smishing attacks
To provide social media with a mechanism to detect phishing messages.
To detect smishing attempts in different languages.
To empower users with proactive protection against financial and personal losses caused by smishing attacks.





Introduction

The use of mobile devices and text messaging has revolutionized communication, but it has also brought about a significant security concern known as smishing attacks. These attacks involve fraudulent text messages that aim to gather sensitive data, taking advantage of the widespread adoption of mobile devices. Smishing attacks are becoming more frequent and sophisticated. To address this growing threat, machine learning methods provide a defense against smishing attacks. This research focuses on developing a machine learning-based approach using techniques like natural language processing, anomaly detection, and pattern recognition to detect and prevent smishing attacks through a user-friendly mobile app. By analyzing the content and context of text messages, machine learning algorithms can identify patterns and anomalies associated with smishing attacks, enabling real-time monitoring and detection. Users will be empowered to recognize and report suspicious messages, enhancing overall user security in the digital environment. The goal of this project is to leverage machine learning techniques within a mobile application to combat smishing attacks and contribute to ongoing efforts to improve user security against emerging cyber threats.

Research Overview

This research focuses on combating the increasing threat of smishing attacks in Sri Lanka. Smishing refers to fraudulent text messages that trick recipients into sharing personal information or engaging in criminal activities. The widespread use of mobile devices in Sri Lanka has made it a prime target for hackers, necessitating robust solutions beyond traditional methods like antivirus software. The research aims to develop a machine learning-based approach to analyze smishing attack characteristics, enabling reliable identification and categorization of text messages. The objective is to create a user-friendly mobile application that utilizes machine learning techniques for detecting and preventing smishing attacks. The research also includes investigating current prevention measures, identifying effective strategies, integrating the model into the mobile application, and exploring language detection capabilities. Ultimately, the goal is to enhance cybersecurity awareness, protect individuals and organizations from smishing attacks, and contribute to a safer online environment in Sri Lanka.

Methodology

Smishing attacks are challenging to handle using traditional rule-based strategies due to their unpredictable nature. To address this, the project adopts a machine learning-based approach, leveraging data-driven models to enhance the accuracy and adaptability of smishing detection. By training the models on relevant datasets, the system can learn patterns and characteristics of smishing messages, enabling more effective recognition and response to these attacks.





The study utilizes a combination of software and hardware elements for conducting experiments. Android emulators and mobile devices running Android API 33 rules and Android 13 OS were employed to simulate real-world mobile environments. This ensures that the developed solution is compatible with the targeted devices and reflects the behavior of actual users.

The dataset used in the project was processed using www.kaggle.com, a popular platform for data science and machine learning. This platform provides a wide range of datasets and tools for preprocessing and analyzing data. By utilizing Kaggle, the project benefits from a diverse dataset and can leverage the platform's resources and community for enhanced data processing capabilities. To create an API containing the trained machine learning model, the project leverages Azure Cloud. Azure Cloud offers a robust and scalable infrastructure for deploying and managing machine learning models in a cloud environment. By utilizing Azure, the project ensures that the trained model can be accessed and utilized by the mobile application in a reliable and efficient manner.

The choice of Android API 33 and Android 13 OS for experimentation enables the project to target a specific version of the Android operating system. This allows for more focused testing and validation of the smishing detection system on the desired platform. By aligning with the targeted Android versions, the project aims to ensure compatibility and optimal performance of the solution on real mobile devices running these operating systems.
